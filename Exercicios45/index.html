Crie um array de objetos com o seguinte conteúdo: [{name:'Aegir', power:10}, {name:'Aud', power: 5} , {name:'Balder', power:7}, {name:'Bragi', power:4}, {name: 'Búri', power:9}, {name: 'Dag', power:10}, {name:'Dellingr', power: 3}].

  1. Crie também um input de texto e um botão, que conforme o que o usuário digita e aperta no botão de confirmação irá aparecer os deuses que contém aquele caractere no nome em uma lista.
  2. Crie também um select com duas opções, sendo: “do maior para o menor” e “do menor para a maior” em que conforme o usuário seleciona ele ordena os deuses através do poder e mostra em uma lista.
  3. Crie também dois inputs de texto, um para o nome e outro para poder e um botão. Será usado para cadastrar novos deuses, e conforme o usuário aperta no botão vai inserindo um novo deus na lista.
  4. Mostre os deuses em uma lista com um botão ao lado de cada deus escrito “deletar”. Conforme o usuário aperta no botão os deuses precisam sumir da lista.
